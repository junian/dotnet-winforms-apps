# Bouncing Ball Simulation

![Bouncing Ball][demo00]

Bouncing Ball Simulation Implemented using C++.NET.

Features:
- Colorful balls
- 3 kind of size
- Multiple balls
- Ball to ball collision
- Reverse gravity
- Slow Motion
